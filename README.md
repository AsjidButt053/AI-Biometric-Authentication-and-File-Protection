# 🔐 AI-Based Biometric Authentication & File Protection System

A **GUI-based Biometric Authentication and File Protection System** developed in **Python** as a **Semester 5 Software Engineering project**. The application combines **Artificial Intelligence**, **Computer Vision**, and **AES Fernet Encryption** to provide secure biometric authentication and protect sensitive files from unauthorized access.

---

## 📖 Project Overview

This project is designed to enhance file security using biometric authentication. Instead of relying solely on passwords, users are authenticated through AI-powered face recognition. After successful verification, users can securely encrypt and decrypt files using the AES Fernet encryption algorithm.

The system integrates Artificial Intelligence, Computer Vision, Cryptography, and GUI development to provide a secure and user-friendly desktop application.

## 💡 Workflow

1. Launch the application.
2. Register a new user with facial data.
3. Authenticate using face recognition.
4. Select a file for encryption.
5. Encrypt the file using AES Fernet.
6. Authenticate again when needed.
7. Decrypt the protected file after successful biometric verification.
